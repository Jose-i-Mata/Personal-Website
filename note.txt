Flexbox is the concept of using a grid like layout to distribute and organize items in a space.

Flex Container - The overarching area that is laid out within a document to house the content.

Flex Item - Items or objects that are contained within the flex container area.

justify-content - How we align the flex items along the main axis.

align-item - How we align the flex items along the cross axis.

align-items - *Only takes into effect when the single row axis (either main axis or cross axis) must be slip into multiple rows
when the size of the flex container is not large enough for the flex items along the row axis* How we align rows with respect to each other in the flex container.
